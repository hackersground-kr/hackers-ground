# 클라우드 스킬 챌린지 완주 인증 안내

안녕하세요, 해커그라운드 참가자 여러분!

클라우드 스킬 챌린지를 성공적으로 완료하신 것을 진심으로 축하드립니다. 이제 여러분의 완주를 인증하는 절차에 대해 안내해 드리겠습니다. 이 과정은 여러분이 사전워크샵에 참여할 자격을 갖추는 중요한 단계입니다.

## 완주 인증 절차
1. **GitHub 이슈 제출**: 완주하신 클라우드 스킬 챌린지 (AI-900, AZ-900)에 대한 [GitHub 이슈를 생성](https://github.com/hackersground-kr/hackers-ground/issues/new/choose)해 주세요. 이슈 제목과 본문에는 각 챌린지 코드와 완료한 내용을 명확하게 기술해야 합니다.
   <div style="text-align: center;">
      <img src="https://github.com/hackersground-kr/hackers-ground/blob/main/templates/images/csc_issue.jpg?raw=true" alt="클스챌 완주 이슈 예시" width="100%"/>
    </div>
2. **인증 프로세스**:
    - 이슈가 생성되면, 자동으로 저희 시스템에서 검증 과정이 시작됩니다.
    - 이슈 내용은 GitHub ID, Microsoft Learn 프로필 링크 등이 정확해야 합니다. 오류가 있는 경우, 적절한 피드백과 함께 수정 요청이 이루어집니다.

3. **라벨 부여**:
    - **완료(COMPLETED)**: 모든 요구사항이 충족되고, 이슈 내용이 정확한 경우, 'verified' 라벨이 부여되며, 참가자는 사전워크샵 참여 자격이 부여됩니다.
    - **미완료(NOT_COMPLETED)**: 요구된 챌린지를 모두 완료하지 않은 경우, 'incomplete' 라벨이 부여되고, 수정을 요청받게 됩니다.
    - **오류(INVALID)**: 제출한 이슈의 정보가 잘못된 경우, 'invalid' 라벨이 부여되며, 이슈는 자동으로 닫힙니다.

4. **해결 및 피드백**:
    - 각 상황에 맞는 적절한 피드백을 제공합니다. 예를 들어, 제출 마감일을 넘긴 경우 또는 GitHub 프로필 주소가 유효하지 않은 경우 등에 대한 안내가 이루어집니다.
    - 정확한 정보 제출과 문제 해결을 위한 지원이 제공됩니다.

5. **Discord 알림**: 챌린지 완료 인증이 끝나면, Discord 채널에 성공적인 완주를 축하하는 메시지가 올라옵니다.


#### 주의사항
- 완주 인증을 위해 제출한 GitHub 이슈는 정확하고 완전해야 합니다. 이슈 제출 후, 꼭 확인사항을 다시 한 번 검토해 주세요.
- 완주 인증 과정 중 발생할 수 있는 오류에 대비하여, 제출 전 모든 정보를 정확히 입력하는 것이 중요합니다.

- 이 시간이 지나서 제출한 이슈는 자동으로 제출기한 초과로 닫힙니다.
