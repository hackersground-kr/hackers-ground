# 해커그라운드 해커톤

## 주제 

* 대구/경북 지역의 사회 문제를 해결할 수 있는 서비스
    * (가산점 사항) 경북 의성군 대상 서비스

## 사용 기술스택

* Microsoft Azure 클라우드 기반
    * PaaS 또는 서버리스 앱 서비스
    * 웹 기반 앱 서비스
    * 모바일 **"웹"** 앱
    * PaaS 또는 서버리스 데이터베이스
    * Table Storage, Cosmos DB, Azure SQL 또는 PostgreSQL
* (가산점 사항) 클라우드 네이티브 앱 개발시 
    * Azure Container Apps
